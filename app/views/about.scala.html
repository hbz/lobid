About

lobid is the North Rhine-Westphalian Library Service Center's (hbz) Linked Open Data service.
The akronym 'lobid' stands for "linking open bibliographic data". lobid provides easy access to various kinds of library data:

  • Data from the hbz union catalog is available via the /resource and /item API endpoints.
  • Authority data from the Integrated Authority File (GND) is available in whole via the /subject endpoint and in part via the /person endpoint.
  • Data from the German ISIL registry and from the MARC Code List for Organizations is made available via the /organisation endpoint.

License and Provenance

The /resource and /item endpoints provide access to data from the hbz union catalogue that is published under CC0. The data is updated on a daily basis. See the data hub for downloading a base dump and updates: Raw data export in Aleph MAB XML, lobid RDF. For downloading large portions of lobid data see also the documentation of the scroll parameter.

The /person and /subject endpoints provide access to data which is published by Deutsche Nationalbibliothek (German National Library, DNB) under CC0. The data is updated on a daily basis. For downloading the whole dump see https://www.dnb.de/EN/Service/DigitaleDienste/LinkedData/linkeddata_node.html.

The /organisation endpoint provides access to data from the German ISIL registry as well as data from the MARC Code List for Organizations. Though the latter is considered as being in the public domain, the German ISIL data is NOT (yet) openly licensed. That is why until now no full dumps are provided anywhere.

The /searchendpoint enables access to all data sources at once and thus doesn't necessarily provide data that is openly licensed.
